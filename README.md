## Installation

`helm-orgcard` is available on [MELPA][melpa-link] and [MELPA-STABLE][melpa-stable-link].

You can install `helm-orgcard` with the following command.

<kbd>M-x package-install [RET] helm-orgcard [RET]</kbd>
